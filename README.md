## 📂 Overview of Graduate ML Work

My graduate coursework and projects span the full progression of modern machine learning:

- **Supervised Learning** — Regression, classification, regularization, trees, ensembles
- **Unsupervised Learning** — Clustering, PCA, probabilistic mixture models
- **Deep Learning** — CNNs, sequence models, attention, transfer learning
- **Natural Language Processing** — Language modeling, embeddings, text classification

Each project emphasizes:

- Clean experimental design  
- Proper train/validation/test separation  
- Cross-validation and hyperparameter tuning  
- Metric-driven evaluation  
- Bias–variance analysis  
- Model interpretability  
- Reproducibility and documentation
