## 📂 Overview of Graduate ML Work

My graduate coursework and projects span the following core areas:

- Supervised Learning (regression, classification, evaluation)
- Unsupervised Learning (clustering, PCA, exploratory analysis)
- Deep Learning (neural networks, optimization, regularization)
- Natural Language Processing (statistical language models)

Each project emphasizes:
- Clean experimental design
- Proper evaluation and interpretation of results
- Generalization behavior and model limitations
- Reproducibility and clear documentation

## 🧪 Project & Lab Summary

### 🔹 Supervised Learning
- **Lab 01 – Supervised Learning Foundations**
  - Logistic regression and neural network classifiers
  - Feature scaling, train/validation/test splits
  - Performance comparison using standard metrics

- **Lab 02 – Regression & Model Evaluation**
  - Linear and regularized regression models
  - Bias–variance tradeoffs
  - Error analysis and model comparison

---

### 🔹 Unsupervised Learning
- **Lab 01 – USArrests Exploratory Analysis**
  - Distance metrics and clustering intuition
  - Data normalization effects on similarity

- **Lab 02 – Principal Component Analysis (PCA)**
  - Dimensionality reduction
  - Variance explained analysis
  - Visualization of high-dimensional structure

---

### 🔹 Deep Learning
- **Lab 01 – First Neural Network in Keras**
  - Feedforward neural network built from scratch
  - Binary classification on real-world data
  - Training curve visualization and interpretation

- **Lab 02 – Regularization & Generalization (Fashion-MNIST)**
  - L2 regularization
  - Dropout
  - Batch normalization
  - Early stopping
  - Comparative analysis of generalization performance

---

### 🔹 Natural Language Processing
- **Lab 01 – N-gram Language Modeling**
  - Unigram, bigram, and trigram models
  - Laplace smoothing
  - Conditional probability estimation
  - Perplexity-based evaluation
